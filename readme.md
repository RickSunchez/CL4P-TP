# Трепло

Простой бот для размножения сообщений

Алгоритм прост:
1. запускаем бота
2. добавляем группы или каналы, важно:
    1. группа или канал должны быть публичными
    2. бот должен быть в администраторах
3. создаем пост
4. profit!
